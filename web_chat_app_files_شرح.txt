=========================================
شرح مشروع لقانا الأردن شات - للمطور
=========================================

نبذة عن المشروع:
-----------------
تطبيق دردشة (Chat Application) مبني بتقنيات الويب (HTML/React/Firebase) وتم تحويله إلى تطبيق Android باستخدام Flutter WebView.

هيكل المشروع:
-------------
1. التطبيق الأصلي (Web):
   - index.html: التطبيق الرئيسي للمستخدمين
   - admin.html: لوحة التحكم للإدارة

2. تطبيق Flutter (الإطار):
   - lib/main.dart: ملف Flutter الرئيسي
   - assets/html/: مجلد يحتوي على نسخة من ملفات الويب
   - pubspec.yaml: ملف التبعيات

كيفية بناء التطبيق:
--------------------
الطريقة الحالية: استخدام GitHub Actions لبناء APK تلقائياً

1. التعديلات على ملفات الويب:
   - عدل ملف web_chat_app/index.html
   - انسخ الملف إلى assets/html/index.html
   - ارفع التغييرات إلى GitHub

2. بناء APK:
   - بعد الرفع، GitHub Actions ستبني APK تلقائياً
   - اذهب إلى: https://github.com/OV-hiroki/lqana-jordan-chat/actions
   - انتظر اكتمال البناء (5-10 دقائق)
   - حمل ملف app-release.apk

المشاكل الحالية:
----------------
1. البطء (Performance):
   - التطبيق بطيء جداً
   - السبب المحتمل: الحركات المعقدة (animations) في CSS
   - الموقع: في index.html

2. إرسال الصور (Image Upload):
   - ميزة إرسال الصور لا تعمل بشكل صحيح
   - السبب: دالة uploadImageExternal تستخدم API خارجي (imgbb)
   - الموقع: في index.html (السطور 434-453 تقريباً)

3. التنسيق (Layout):
   - تم إصلاحها: تم تقليل ارتفاع header من h-14 إلى h-12
   - تم تقليل z-index من z-20 إلى z-10

الملفات المطلوب إصلاحها:
------------------------
- web_chat_app/index.html فقط (يحتوي على كل الكود: React, CSS, JS, Firebase)

التقنيات المستخدمة:
-------------------
- React (للواجهة)
- Tailwind CSS (للتنسيق)
- Firebase (للقاعدة البيانات والمصادقة)
- PeerJS (للمكالمات الصوتية)
- Flutter WebView (لتحويل الويب إلى Android)

ملاحظات مهمة:
-------------
- لا توجد ملفات CSS أو JS منفصلة (كل شيء مدمج في HTML)
- التطبيق يعمل بشكل صحيح في المتصفح
- المشاكل تظهر فقط في تطبيق Android (WebView)
- Flutter WebView يحافظ على الشكل الأصلي للويب

للمطور:
--------
يرجى التركيز على:
1. تحسين الأداء في index.html (تقليل الحركات المعقدة)
2. إصلاح دالة uploadImageExternal أو استخدام بديل
3. التأكد من أن التعديلات لا تؤثر على الشكل في المتصفح

بعد الإصلاح:
-------------
1. عدل web_chat_app/index.html
2. انسخ الملف إلى assets/html/index.html
3. ارفع التغييرات إلى GitHub
4. انتظر بناء APK الجديد
5. اختبر التطبيق
